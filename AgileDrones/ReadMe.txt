README


1. Download FlightGoggles_v1.5.6_stable_Windows64.tar.gz from the following link and place it in this directory

https://github.com/AgileDrones/FlightGoggles/releases/download/untagged-cfc4db08b42e68a4c1cf/FlightGoggles_v1.5.6_stable_Windows64.tar.gz

2. Extract the tar.gz file using the following command (in cmd prompt)
tar -xzvf FlightGoggles_v1.5.6_stable_Windows64.tar.gz -C ./FlightGoggles

3. Run FlightGoggles_v1.5.6_stable_64.exe (found in the extracted data)

4. Open ./OfflineRenderClient path in matlab. Run "run_offline_client.m" script in matlab.
